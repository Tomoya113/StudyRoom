# StudyRoom
HackU2020 YAKITORI.rbのリポジトリです。
# 使い方
```
// イメージのビルド
$ docker-compose build
(毎回はいらないかも)

// コンテナの起動
$ docker-compose up -d

// ログを見たい時
$ docker-compose logs -f app

// コンテナに入る
$ docker-compose exec app sh
```
